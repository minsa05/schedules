# Schedule & Comment Management API

스프링 부트(Spring Boot) 기반의 일정(Schedule)·댓글(Comment) 관리 RESTful API 프로젝트입니다.  


## 주요 기능

- **일정 관리 (Schedule)**
  - 생성, 전체 조회, 단일 조회, 수정, 삭제
- **댓글 관리 (Comment)**
  - 일정별 댓글 목록 조회, 댓글 생성, 단일 댓글 조회, 수정, 삭제
- **대댓글 관리 (Reply)**
  - 댓글에 대한 대댓글(1Depth) 생성, 대댓글 목록 조회, 수정, 삭제
